# CarrotCMS

# DE HUIDIGE VERSIE IS NOG NIET KLAAR. DE DATABASE SYSTEMEN ZIJN OOK NOG NIET KLAAR. DIT IS ALLEEN EEN VOORPROEFJE.

Het is simpel te installeren, heel makkelijk te customizen en nog veel meer!
